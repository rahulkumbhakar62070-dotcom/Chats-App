# 💬 WhatsApp Chat CRUD Application

A simple WhatsApp-inspired Chat CRUD Application built using **Node.js**, **Express.js**, **MongoDB**, **Mongoose**, and **EJS**.

---

## 🚀 Features

- ➕ Create New Chat
- 📋 View All Chats
- ✏️ Edit Existing Messages
- 🗑️ Delete Chats
- 📅 Display Date & Time
- 🎨 Responsive UI
- 🍃 MongoDB Integration
- ⚡ RESTful CRUD Operations

---

## ⚙️ Installation

### Clone the repository

bash
git clone https://github.com/yourusername/whatsapp-chat-crud.git


### Move into the project

bash
cd whatsapp-chat-crud


### Install dependencies

```bash
npm install
```

### Start MongoDB

Make sure MongoDB is running locally.

### Run the project

```bash
node index.js
```

Open your browser:

```
http://localhost:8080/chats
```
